- Word of the Day
Is a clean and simple web application that helps you discover and learn a new word every day. This project fetches a random word, displays its definition, and can even pronounce it for you.

- Based on the app, here's what you can do:
* Discover Words: See a "Word of the Day" with its definition.
* Get a New Word: Click the "New Word" button to fetch a new random word and definition at any time.
* Hear Pronunciation: Click the "Speak" button to use the browser's built-in text-to-speech (TTS) to hear the word pronounced.
* Clean Interface: A minimal, focused design so you can concentrate on the vocabulary
